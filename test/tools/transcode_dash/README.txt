transcode_dash/transcode.py
---------------------------

This tool is for generating segments for DASH manifests.

DASH manifest(s) are obtained by harvesting See ../harvest/README.txt or via 
other means

Example Usage.
--------------

1. Harvest a stream and transcode all segments

Since the segments will all be generated by transcode the following harvest option 
may be used
.../harvest/harvest.py --no_segments ....

.../transcode_dash/transcode.py --all

2. Harvest a 30min cloud TSB for 5 mins and only transcode the segments from the last 5mins

.../harvest/harvest.py --no_segments

We look at the first DASH manifest from the harvest I.E .../mpd.1 and find the last period
<Period id="1728912086689-1" start="PT480253H21M27.367S">
Use the start value so that only segments from that period and subsequent are transcoded
.../transcode_dash/transcode.py --all --start_at PT480253H21M27.367S

3. A single dash manifest has been created and there is no harvest_details.json because it was
 not created by harvest.py. We want to generate the segments for that.

.../transcode_dash/transcode.py --all --manifest path/manifest.mpd

exists:
path/manifest.mpd

4. A sequence of live manifests exist but there is no harvest_details.json. Manifests ending in 
a number will be read and transcoded.

.../transcode_dash/transcode.py --all --manifest path/manifest.mpd

exists:
path/manifest.mpd.1
path/manifest.mpd.2
..
but importantly no path/manifest.mpd


Requirements
-------------

ffmpeg ver 7+ is required. For Ubuntu it may be installed from 
https://ubuntuhandbook.org/index.php/2024/04/ffmpeg-7-0-ppa-ubuntu/
